# BukuTask MVP

This package contains a working front-end marketplace prototype.

## Open it

Double-click `index.html` or upload all files to any static web host.

## Included

- Responsive landing page
- Job search and category filtering
- Job posting form
- Two-post rolling 24-hour free-plan limit
- Local browser storage for newly posted jobs
- Premium pricing page
- PayPal support section
- Reviews and safety sections
- Draft Terms, Privacy, Community, and Prohibited Services policies
- Social login and identity/payment integration placeholders
- Responsive mobile design

## Important before public launch

This is a front-end MVP, not a production marketplace. Before allowing real users:

1. Connect authentication and database storage.
2. Add verified email and phone workflows.
3. Integrate a professional ID-verification provider.
4. Connect a PayPal Business subscription plan.
5. Add image moderation and secure private uploads.
6. Add in-app messaging and reporting.
7. Add an admin moderation dashboard and audit log.
8. Have a licensed attorney review all policies.
9. Replace the temporary PayPal support URL with your official PayPal.me or donation link.
10. Add abuse monitoring, rate limiting, backups, and security logging.

## Suggested production stack

Next.js, Supabase/PostgreSQL, Supabase Auth or Clerk, Cloudinary or S3, Stripe Identity or Persona, PayPal Subscriptions, Resend, Twilio, Cloudflare, and Sentry.
